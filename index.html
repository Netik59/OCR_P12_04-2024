Plan the development of a website

This project was not to code but to prepare and plan the development of the website. Indeed, I established a technical monitoring system on Feedly, organized according to two axes: the specific aspects of the project and the general trends of web development. This monitoring allowed me to start listing technical tools that could facilitate the development of the site. I also identified the technical challenges of the project by grouping the needs, structuring my ideas and considering technical solutions for each aspect of development. In parallel, I set up a Kanban board on Trello to manage the project tasks, making sure to assign each task, estimate its complexity and detail its technical specifications.
